# Default Metrics

[!UICONTROL **Edit Settings**] > [!UICONTROL **Individual Report Suite Settings**] > [!UICONTROL **Default Metrics**]

Analytics displays a default (out-of-the-box) set of metrics in all reports and projects, unless a user selects a custom set of metrics. The selected metrics display for all users of the associated report suite. You can update only one report suite's default metrics at a time. These settings do not guarantee the order in which the default metrics are displayed.

For more information, see: 

* [Metrics in Analysis Workspace](/help/analyze/analysis-workspace/components/apply-create-metrics.md)

* [Calculated Metrics](/help/components/calculated-metrics/cm-overview.md)
