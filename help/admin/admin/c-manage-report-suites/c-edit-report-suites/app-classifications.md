# App classifications

Mobile classifications are enabled after you enable any of the features under [App reporting](app-reporting.md). It provides an identical interface to [Conversion classifications](conversion-var-admin/conversion-classifications.md) or [Traffic classifications](c-traffic-variables/traffic-classifications.md), but for app reporting dimensions.

**[!UICONTROL Analytics]** > **[!UICONTROL Admin]** > **[!UICONTROL Report Suites]** > **[!UICONTROL Edit Settings]** > **[!UICONTROL App Management]** > **[!UICONTROL App Classifications]**

Adobe recommends instead using [Classification sets](/help/components/classifications/sets/overview.md) for all new classification data.
