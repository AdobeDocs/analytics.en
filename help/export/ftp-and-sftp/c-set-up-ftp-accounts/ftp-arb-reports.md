# Report Builder delivered reports

Use the Adobe Report Builder add-in for Microsoft Excel to build customized requests from Web analytics data, which you can insert into your Excel worksheets.

Requests can dynamically reference cells within your worksheet, and you can update and customize how the data is presented. For FTP or SFTP, you must ensure that the host is available externally. For more information on the [!UICONTROL Report Builder] delivered reports, see [Report Builder Help](/help/analyze/report-builder/rb-overview.md).

For information on FTP or SFTP limits and data retention, see [FTP and SFTP Limits and Data Retention](/help/export/ftp-and-sftp/ftp-limits.md).
