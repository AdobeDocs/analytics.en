# Code Manager

Code manager lets you download data collection code for web and mobile platforms.

**[!UICONTROL Analytics > Admin > Code Manager]**.

After you download the library, you must configure the code to send data to the correct tracking server and report suite. Additional implementation resources are available at [Developer & Implementation](https://marketing.adobe.com/resources/help/en_US/reference/developer.html).
