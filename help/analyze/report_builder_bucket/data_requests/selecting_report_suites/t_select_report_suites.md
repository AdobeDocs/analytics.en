# Select a report suite


>1. In Excel, click **[!UICONTROL  Add-Ins]**.
>1. In the toolbar, click **[!UICONTROL  Create]**.
>1. In the [!UICONTROL  Request Wizard: Step 1] window, select a report suite under [!UICONTROL  Report Suite].
