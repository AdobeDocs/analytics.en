# Create a Data Warehouse request

Steps that describe how to create a Data Warehouse request.

1. Navigate to **[!UICONTROL Analytics]** > **[!UICONTROL Tools]** > **[!UICONTROL Data Warehouse]**.
1. On the [!UICONTROL Data Warehouse Request] tab, configure the options described in [Data Warehouse Requests Descriptions](../../export/data-warehouse-bucket/data-warehouse.md#section_F21C78ED36884C389C852E876AF5CDE8)
1. Click **[!UICONTROL Request this Report]**.
