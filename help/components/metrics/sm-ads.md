# Streaming media services ad metrics

Streaming media services ad metrics provide supplemental reporting functionality for data collected through streaming media collection libraries. These metrics require the **[!UICONTROL Adobe Analytics for Streaming Media Add-on]**. Contact your Adobe Account Team for details.

To use these metrics, enable **[!UICONTROL Media Ads]** under [[!UICONTROL Media Reporting]](/help/admin/tools/manage-rs/edit-settings/media-management.md) for the report suite.

The following metrics are available:

* [[!UICONTROL Ad completes]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/ad-completes)
* [[!UICONTROL Ad starts]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/ad-starts)
* [[!UICONTROL Ad time spent]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/ad-time-spent)
* [[!UICONTROL Media time spent]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/media-time-spent)

See [Streaming media services ad dimensions](../dimensions/sm-ads.md) for corresponding dimensions.
