# Device graph

>[!WARNING]
>
>Device Graph within Cross-Device Analytics is [discontinued](https://experienceleague.adobe.com/en/docs/discontinued/using/device-graph) and no longer available as of **December 31, 2025**. Please switch any current device graph enabled virtual report suite to the [field-based method](/help/components/cda/field-based-stitching.md).
