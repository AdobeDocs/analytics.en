# Report Builder overview

 Report Builder is an add-in for Microsoft Excel. It lets you build customized requests from reporting and analytics data, which you can insert into your Excel worksheets. Requests can dynamically reference cells within your worksheet, and you can update and customize how Report Builder presents the data.
